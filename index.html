    5로그인이 되면 보이는 화면

    아래 같은 화면이 뜨면 성공입니다.

    • 인사말 — 본인 이름으로 인사가 떠요 (예: “백암초님”)
    • 입력창 — 여기에 질문을 적으면 됩니다
    • Sonnet 4.6 — 현재 사용 중인 AI 모델 이름
    • 버튼들 — 코드 / 작성하기 / 학습하기 / 일상생활 주제별 추천

    6왼쪽 메뉴바 기능

    화면 왼쪽에 이 메뉴가 보입니다. 안 보이면 좌측 상단 아이콘을 눌러 펼치세요.

    • 새 채팅 — 새 대화를 시작합니다
    • 검색 — 예전에 했던 대화를 찾을 때
    • 채팅 — 지금까지 한 대화 목록
    • 프로젝트 — 자료를 모아두고 길게 작업할 때 사용
    • 코드 — 코딩 관련 작업 공간
    • 사용자 지정 — 내 스타일로 맞춤 설정
    • Design — 디자인 작업용
    • 더보기 — 그 외 기능들
