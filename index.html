2클로드 로그인

이 화면이 뜨면 로그인 페이지에 잘 들어온 거예요.

두 가지 방법 중 하나를 고르면 됩니다.

  • ① Google로 계속하기 — 가장 빠릅니다 (아래 3번 참고)
  • ② 이메일로 계속하기 — 이메일을 입력하면 인증 코드가 옵니다

3구글 로그인

로그인 화면에서 “Google로 계속하기” 버튼을 누르고, 아래 백암초 계정 중 하나로 로그인하세요.

계정
비밀번호
yipa0301@goe.go.kr qordka1320^^
yipa0302@goe.go.kr qordka1320^^
yipa0303@goe.go.kr qordka1320*

계정/비번을 클릭하면 자동 선택됩니다. 옆 “복사” 버튼을 누르면 한 번에 복사돼요.

4매번 로그인이 어렵다면? 데스크톱 앱

로그인 화면 맨 아래의 “데스크톱 앱 다운로드”를 눌러 설치하세요.
한 번 로그인해두면 그 다음부터는 바로 열립니다.

💡 설치 후엔 바탕화면에 Claude 아이콘이 생겨요. 더블클릭만 하면 끝.

5로그인이 되면 보이는 화면

아래 같은 화면이 뜨면 성공입니다.

  • 인사말 — 본인 이름으로 인사가 떠요 (예: “백암초님”)
  • 입력창 — 여기에 질문을 적으면 됩니다
  • Sonnet 4.6 — 현재 사용 중인 AI 모델 이름
  • 버튼들 — 코드 / 작성하기 / 학습하기 / 일상생활 주제별 추천

6왼쪽 메뉴바 기능

화면 왼쪽에 이 메뉴가 보입니다. 안 보이면 좌측 상단 아이콘을 눌러 펼치세요.

  • 새 채팅 — 새 대화를 시작합니다
  • 검색 — 예전에 했던 대화를 찾을 때
  • 채팅 — 지금까지 한 대화 목록
  • 프로젝트 — 자료를 모아두고 길게 작업할 때 사용
  • 코드 — 코딩 관련 작업 공간
  • 사용자 지정 — 내 스타일로 맞춤 설정
  • Design — 디자인 작업용
  • 더보기 — 그 외 기능들
